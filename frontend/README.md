# CBC E-Bookshop & Learning Hub

A comprehensive, full-stack web application tailored for Kenya's Competency-Based Curriculum (CBC). This platform bridges the gap between parents, schools, and book vendors by offering a seamless way to purchase new learning materials, save up for upcoming terms, resell used textbooks, and view school-specific booklists.

---

## 🚀 Key Features

* **📚 Primary Marketplace:** Browse and purchase CBC-approved books, stationery, and learning materials online with seamless **M-Pesa (Daraja API)** integration.
* **♻️ Peer-to-Peer Reselling:** A dedicated secondhand marketplace for parents and students to list and resell used textbooks, promoting sustainability and affordability.
* **💰 Term Savings & Installments:** A built-in wallet/financial planning tool allowing users to lock funds or pay in installments to prepare for next term's educational expenses.
* **🏫 Institutional School Lists:** Schools can manage profiles and upload curriculum-specific booklists for each grade, allowing parents to buy exactly what their children need in a single click.
* **📊 Admin & School Portals:**
    * *Admin Dashboard:* Track business insights, manage product inventory, and monitor platform transactions.
    * *School Management:* Verified schools can add, update, and manage their required textbook listings per grade.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3, Tailwind CSS, JavaScript (Framework: React / Next.js)
* **Backend:** Python 3.12 (Flask / Django FastAPI)
* **Database:** PostgreSQL / MySQL / MongoDB
* **Payment Integration:** Safaricom M-Pesa (Daraja API)
